Task 与异步编程(async/awiat)详细学习
2021-01-07

Task

一个 Task 表示没有返回值通常以异步执行的一个操作。 Task 通常在线程池线程上异步执行，而不是在主应用程序线程上同步执行，它可以使用 Status IsCanceled IsCompleted 和 IsFaulted 属性来确定任务的状态。对于有返回值的操作可使用 Task<TResult> 类。

Task 最常见的方法使用其静态方法 Run 来启动一个任务，其返回一个 Task 任务的对象

也可用 Task 的 Factory.StartNew 来创建 Task 对象，其允许执行带有参数的 Action。

Task.Run 或 Task.Factory.StartNew 方法创建的任务会立即执行，如需要创建任务后在需要的时候去执行可使用构造函数方式创建 Task 任务。

命名约定

返回常规可等待类型的方法（例如 Task、Task<T>、ValueTask 和 ValueTask<T>）应具有以“Async”结束的名称。 启动异步操作但不返回可等待类型的方法不得以具有“Async”结尾的名称。
