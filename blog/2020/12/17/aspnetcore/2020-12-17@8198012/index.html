ASP.NET Core 简介

ASP.NET Core 是一个跨平台的高性能开源框架，使用 ASP.NET Core 可以

  • 开发 Web 应用和服务、物联网 (IoT) 应用和移动后端。
  • 在 Windows、macOS 和 Linux 上进行开发。
  • 部署到云或本地。
  • 在 .NET Core 上运行。

详细请参考

先决条件

安装 .NET Core SDK

一般在安装 Visual Studio 开发工具时勾选即可自动安装。

技术介绍

ASP.NET Core 相较于 ASP.NET 有一些结构上的变化，ASP.NET Core 是一个更精简的、模块化的框架。ASP.NET Core 不再基于 System.Web.dll，当前它基于一系列颗粒化拥有良好构建的 NuGet 包。

ASP.NET Core 应用其实就是一个在其 Main 方法中创建一个 Web 服务器的简单控制台应用程序。

可以新建一个 ASP.NET Core 项目查看其 Program 类
