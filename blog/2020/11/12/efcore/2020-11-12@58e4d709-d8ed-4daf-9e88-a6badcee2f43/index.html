基于链式方法的查询模式

1. 筛选 Where

  • 通过构建 Lambda 表达式进行查询。

2. 获取单个元素 First Last

  • 我们可以看到 Where 的返回值类型为 IQueryable<T> 而这个接口类型就是 LINQ 查询的基础，那么我们还可以通过此类型的数据集继续进行操作。
  • 通过 First 方法我们可以获取集合中的第一条数据，当集合中的数据数量为0的时候，此方法会抛出一个异常。而 FirstOrDefault 方法在这种情况下会返回一个 null 值，我以我们在用的时候经常会用 FirstOrDefault 的方式获取，这样仅仅是少了一次异常捕获，多了一次 null 值判断。
  • Last 与 First 用法相同，仅仅是从获取第一条记录变成了获取最后一条。

无跟踪查询

无跟踪查询就是在你查询到数据之后不再对你查询到的数据进行跟踪，次用法在你保证查询出的数据作为只读数据来使用，这样会提升一定的性能，使用的方法也很简单。

  • 在表达式之后调用 AsNoTracking() 即可。
