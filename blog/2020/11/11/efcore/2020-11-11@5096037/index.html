EF Core 简介

EF Core 全称 Entity Framework Core 是轻量化、可扩展、开源和跨平台版的常用 Entity Framework 数据库访问技术。EF Core 可用作对象关系映射(O/RM)。

它不仅仅是一个 O/RM 框架来使用，它的功能要比其他数据库访问框架强大的多，值得去深入学习。

  1. 右键项目 -> 管理 NuGet 程序包

    1. 首先安装 Microsoft.EntityFrameworkCore.Sqlite 包
      • 这个包的是EF的Sqlite支持包，如果你想用SQL Server可以装 Microsoft.EntityFrameworkCore.Sqlserver 这个包。
    2. 然后安装 Microsoft.EntityFrameworkCore.Tools 包
