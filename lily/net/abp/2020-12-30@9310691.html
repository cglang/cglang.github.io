# ABP 中获取当前用户相关的信息

当前用户是指当前请求相关的用户。由于在 Web 程序中当前用户的信息的使用是非常常见的， ABP 对此做了一次封装。

# ICurrentUser

ICurrentUser 是获取当前活动用户信息的主要服务。

通过构造函数注入的方式使用该服务

using System;
using Volo.Abp.DependencyInjection;
using Volo.Abp.Users;

namespace AbpDemo
{
    public class MyService : ITransientDependency
    {
        private readonly ICurrentUser _currentUser;

        public MyService(ICurrentUser currentUser)
        {
            _currentUser = currentUser;
        }
        
        public void Foo()
        {
            Guid? userId = _currentUser.Id;
        }
    }
}

公共基类已将此服务作为基属性注入。比如

using System;
using Volo.Abp.Application.Services;

namespace AbpDemo
{
    public class MyAppService : ApplicationService
    {
        public void Foo()
        {
            Guid? userId = CurrentUser.Id;
        }
    }
}

# 属性(Properties)

  • IsAuthenticated (bool) ：是否已登录认证。
  • Id (Guid?)：获取用户的Id，未登录时返回 null 。
  • UserName (string)：获取用户的名称，未登录时返回 null 。
  • TenantId (Guid?)：获取用户租户的Id，未分配租户时返回 null。
  • Email (string)：获取用户的 Email。
  • EmailVerified (bool)：是否已验证电子邮箱。
  • PhoneNumber (string)：获取用户手机号码。
  • PhoneNumberVerified (bool)：是否已验证手机号。
  • Roles (string[])：获取当前用户拥有的角色名。

# 方法(Methods)

ICurrentUser 是在 ICurrentPrincipalAccessor 基础上对 claims 处理实现的。上述所有属性实际上都是从当前经过身份验证的用户的 claims 中检索得到的。ICurrentUser 对于检索功能做了些封装。

  • FindClaim: 获取具有给定名称的 Claim。如果未找到，则返回 null。
  • FindClaims: 获取具有给定名称的所有 Claim（允许多个具有相同的名称）。
  • GetAllClaims: 获取所有的 Claim。
  • IsInRole: A shortcut method to check if the current user is in the specified role.

除了这些标准方法外，还有一些扩展方法：

  • FindClaimValue: 获取具有给定名称的 Claim 的值，如果找不到则为 null 。它有一个泛型重载将值强制转换为特定类型。
  • GetId: 返回当前用户的Id。如果当前用户没有登录会抛出异常。

# ICurrentPrincipalAccessor

ICurrentPrincipalAccessor 是在需要当前用户的主体时使用的服务。

对于 Web 应用程序，它获取当前用户的 HttpContext 属性。对于非web应用程序，它返回当前主线程.。

通常不需要这个低级别的服务

其接口声明内容也极其简单

public interface ICurrentPrincipalAccessor
{
    ClaimsPrincipal Principal { get; }

    IDisposable Change(ClaimsPrincipal principal);
}

Change 方法用于更改当前用户主体内容

public class MyAppService : ApplicationService
{
    private readonly ICurrentPrincipalAccessor _currentPrincipalAccessor;

    public MyAppService(ICurrentPrincipalAccessor currentPrincipalAccessor)
    {
        _currentPrincipalAccessor = currentPrincipalAccessor;
    }

    public void Foo()
    {
        var newPrincipal = new ClaimsPrincipal(
            new ClaimsIdentity(
                new Claim[]
                {
                    new Claim(AbpClaimTypes.UserId, Guid.NewGuid().ToString()),
                    new Claim(AbpClaimTypes.UserName, "john"),
                    new Claim("MyCustomCliam", "42")
                }
            )
        );

        using (_currentPrincipalAccessor.Change(newPrincipal))
        {
            var userName = CurrentUser.UserName; //returns "john"
            //...
        }
    }
}

# AbpClaimTypes

AbpClaimTypes 是一个静态类，用于定义标准的 Claim 名称，其内大部分引用的 ClaimTypes 类中的常量值。

AbpClaimTypes 是 ABP 定义的，ClaimTypes 是微软定义的。
