# ABP 的依赖注入及相关配置

ABP 的依赖注入是基于微软的依赖注入库的。如果要使用动态拦截器，需要依赖 Autofac 库，目的是为了实现动态代理。

# ABP 特点

# 模块化

ABP 是一个模块化框架，每个模块都需要自己定义自己的服务并在自己的单独模块类中通过依赖注入进行注册。

public class BlogModule : AbpModule
{
    public override void ConfigureServices(ServiceConfigurationContext context)
    {
        // 在此处通过 context.Services 对象注入依赖项
    }
}

# 注册服务

# 约定大于配置的原则

ABP 是使用约定大于配置的原则的，它有以下依赖注入约定。

  • 模块类也就是继承自 AbpModule 的类注册为 Singleton
  • 控制器类（继承 Controller 或 AbpController）被注册为 Transient
  • Razor 页面模型（继承 PageModel 或 AbpPageModel ）被注册为 Transient
  • MVC 视图组件（继承 ViewComponent 或 AbpViewComponent）被注册为 Transient
  • 应用程序服务类（实现 IApplicationService 接口或继承 ApplicationService 类）注册为 Transient
  • 仓储类（实现 IRepository 接口）注册为 Transient
  • 域服务类（实现 IDomainService 接口）注册为 Transient

# 禁用默认约定

通过重写模块类的 PreConfigureServices 方法，设置 SkipAutoServiceRegistration 为 true

public override void PreConfigureServices(ServiceConfigurationContext context)
{
    SkipAutoServiceRegistration = true;
}

如果关闭自动注册功能，应当手动进行服务注册，可通过 Service 对象的拓展方法 AddAssemblyOf 来让ABP依旧按照约定注册服务。

public override void ConfigureServices(ServiceConfigurationContext context)
{
	context.Services.AddAssemblyOf<BlogModule>();
}

# 依赖注入接口约定

  • ITransientDependency 注册为 Transient 生命周期。
  • ISingletonDependency 注册为 Singleton 生命周期。
  • IScopedDependency 注册为 Scoped 生命周期。

# 依赖注入 Attribute 特性约定

使用 DependencyAttribute 配置依赖注入服务，它具有以下属性。

  • Lifetime：注册的生命周期，值为：Singleton，Transient 或 Scoped。
  • TryRegister：设置 true 则只注册以前未注册的服务，实际使用的是 IServiceCollection 的 TryAdd 扩展方法。
  • ReplaceServices：设置 true 则替换之前已经注册过的服务，实际使用的是 IServiceCollection 的 Replace 扩展方法。

ExposeServicesAttribute 用于控制使用指定相关类提供服务

[Dependency(ReplaceServices = true)]
[ExposeServices(typeof(ITaxCalculator))]
public class TaxCalculator: ICalculator, ITaxCalculator, ICanCalculate, ITransientDependency
{
}

如果你未指定要公开的服务，则ABP依照约定公开服务，使用 I+类名 约定。

# 手动注册服务

因为 ABP 的依赖注入是基于微软的依赖注入库的，所以手动注入是相同的。

# 使用注册的服务

解析服务有构造函数注入、属性注入、 IServiceProvider 解析服务三种方式。

# 构造函数注入

直接在构造函数参数中声明即可。

public class TaxAppService : ApplicationService
{
    private readonly ITaxCalculator _taxCalculator;
    public TaxAppService(ITaxCalculator taxCalculator)
    {
	    _taxCalculator = taxCalculator;
    }
}

# 属性注入

属性注入需要依赖 Autofac 库才可进行，并且属性必须为 public 限制。

public class MyService : ITransientDependency
{
	public ILogger<MyService> Logger { get; set; }
}

# IServiceProvider 解析服务

public class MyService : ITransientDependency
{
    private readonly IServiceProvider _serviceProvider;
    public MyService(IServiceProvider serviceProvider)
    {
    	_serviceProvider = serviceProvider;
    }
    public void DoSomething()
    {
    	var taxCalculator = _serviceProvider.GetService<ITaxCalculator>();
    }
}
...
// 或者是通过下面的方式
using (var scope = _serviceProvider.CreateScope())
{
    var service1 = scope.ServiceProvider.GetService<IMyService1>();
    var service2 = scope.ServiceProvider.GetService<IMyService2>();
}
