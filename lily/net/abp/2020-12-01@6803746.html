# ABP 的虚拟文件系统

ABP 的虚拟文件系统是基于 .NET Core 文件系统进行扩展的，虚拟文件系统是相对与文件系统来讲的，让你在程序中对文件操作时不用关系文件的存储位置和存储方式。

# 基础需要

ABP 的文件系统是通过 Volo.Abp.VirtualFileSystem 进行操作的，从 NuGet 中进行安装。

# 开始

新建 类库 项目取名为 TestVfs，安装依赖 Volo.Abp.AspNetCore.Mvc Volo.Abp.VirtualFileSystem。添加 MyResources/Hello.html 文件以及文件夹。

<!DOCTYPE html>

<html lang="en" xmlns="http://www.w3.org/1999/xhtml">
<head>
    <meta charset="utf-8" />
    <title>Hello</title>
</head>
<body>
    <h1>Hello,World!</h1>
</body>
</html>

# 将文件嵌入到程序集

双击 TestVfs 项目标签即可打开

<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>net5.0</TargetFramework>
  </PropertyGroup>

  <ItemGroup>
    <EmbeddedResource Include="MyResources\*" />
  </ItemGroup>
  
</Project>

这种方式将添加 MyResources 目录下所有的文件。

# 将嵌入到程序集的文件注册到虚拟文件系统

添加新类 TestVfsModule

using Microsoft.AspNetCore.Builder;
using Volo.Abp;
using Volo.Abp.AspNetCore.Mvc;
using Volo.Abp.AspNetCore.VirtualFileSystem;
using Volo.Abp.Modularity;
using Volo.Abp.VirtualFileSystem;

namespace TestVfs
{
    [DependsOn(typeof(AbpAspNetCoreMvcModule))]
    [DependsOn(typeof(AbpVirtualFileSystemModule))]
    public class TestVfsModule : AbpModule
    {
        public override void ConfigureServices(ServiceConfigurationContext context)
        {
            Configure<AbpVirtualFileSystemOptions>(options =>
            {
                options.FileSets.AddEmbedded<TestVfsModule>("TestVfs.MyResources", "/MyResources");
            });

            //Configure<AbpAspNetCoreContentOptions>(options =>
            //{
            //    options.AllowedExtraWebContentFileExtensions.Add(".html");
            //    options.AllowedExtraWebContentFolders.Add("/");
            //});
        }

        public override void OnApplicationInitialization(ApplicationInitializationContext context)
        {
            var app = context.GetApplicationBuilder();

            app.UseVirtualFiles();
            app.UseStaticFiles();
            app.UseRouting();

            app.UseConfiguredEndpoints(options =>
            {
                options.MapControllerRoute("app1", "{controller=App1}/{action=Index}/{id?}");
            });
        }
    }
}

添加控制器类 TestVfsController

using Microsoft.AspNetCore.Mvc;
using Microsoft.Extensions.FileProviders;
using System.Threading.Tasks;
using Volo.Abp.AspNetCore.Mvc;
using Volo.Abp.Http;
using Volo.Abp.VirtualFileSystem;

namespace TestVfs
{
    public class TestVfsController : AbpController
    {
        private readonly IVirtualFileProvider _virtualFileProvider;

        public TestVfsController(IVirtualFileProvider virtualFileProvider)
        {
            _virtualFileProvider = virtualFileProvider;
        }

        public async Task<IActionResult> Text()
        {
            var fileInfo = _virtualFileProvider.GetFileInfo("/Hello.html");
            var fileBytes = await fileInfo.ReadBytesAsync();
            return File(fileBytes, MimeTypes.Text.Html);
        }
    }
}

新建 ASP.NET Core Web 项目

# 修改 startup 类

using Microsoft.AspNetCore.Builder;
using Microsoft.AspNetCore.Hosting;
using Microsoft.Extensions.DependencyInjection;
using TestVfs;

namespace VFSTest
{
    public class Startup
    {
        public void ConfigureServices(IServiceCollection services)
        {
            services.AddApplication<TestVfsModule>();
        }

        public void Configure(IApplicationBuilder app, IWebHostEnvironment env)
        {
            app.InitializeApplication();
        }
    }
}

启动Web项目，打开地址 /TestVfs/Text。

现在我们取消 TestVfsModule 中的注释，然后在Web项目中添加 wwwtest/Hello.html 文件夹及文件。

<!DOCTYPE html>

<html lang="en" xmlns="http://www.w3.org/1999/xhtml">
<head>
    <meta charset="utf-8" />
    <title>Hello</title>
</head>
<body>
    <h1>Hello,World!</h1>
</body>
</html>

启动项目访问 /wwwtest/Hello.html
