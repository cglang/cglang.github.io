# 在 Ubuntu 20.04 中安装 Docker

Docker 是一个开放源代码的容器化平台，可以在几乎任何地方运行快速构建、测试和部署应用程序的容器

本教程介绍如何在 Ubuntu 20.04 上安装 Docker

我们用官方 Docker 存储库中安装最新的Docker软件包

# 安装 Docker

更新程序包

安装并添加HTTPS存储库所需的依赖项

sudo apt update
sudo apt install apt-transport-https ca-certificates curl gnupg-agent software-properties-common

使用 curl 命令导入存储库的 GPG 密钥

curl -fsSL https://mirrors.ustc.edu.cn/docker-ce/linux/ubuntu/gpg | sudo apt-key add -

添加 Docker 储存库到系统中

sudo add-apt-repository "deb [arch=amd64] https://mirrors.ustc.edu.cn/docker-ce/linux/ubuntu $(lsb_release -cs) stable"

添加储存库后更新 apt 源索引，并安装最新版 Docker 社区版

sudo apt-get update
sudo apt install docker-ce docker-ce-cli containerd.io

安装完成后使用以下命令查看 Docker 安装版本

docker --version

使用以下命令启动并将 Docker 设为随系统启动自动启动

sudo systemctl start docker
sudo systemctl enable docker

至此 Docker 就已经安装成功

# 设置 Docker 国内镜像源进行加速

由于国内网络问题，从 Docker 官方仓库拉取镜像十分缓慢，我们可以需要配置国内镜像仓库来进行加速

编辑下列命令行所指的文件，写入

sudo vi /etc/docker/daemon.json

{
  "registry-mirrors": ["https://hub-mirror.c.163.com"]
}

提示

这里我们使用的是网易的加速地址进行加速，我们也可以使用阿里云的 Docker 镜像加速服务 (opens new window)

保存退出后重新启动 Docker 服务使配置生效

sudo service docker restart
