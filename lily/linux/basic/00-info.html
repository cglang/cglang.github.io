# Linux 简介

学习 Linux 首先要搞明白 Linux 是什么？他有什么应用？它的优势是什么？

# 什么是 Linux ？

Linux 与 Windows 一样同样是一个操作系统软件，与 Windows 不同之处在于，Linux 是一套开放源代码程序的、可以自由传播的类 Unix 操作系统软件。同时他是一个多用户、多任务的网络操作系统。它是由世界各地成千上万的程序员设计和开发实现的，当初开发 Linux 系统的目的就是建立不受任何商业化软件版权制约的、全世界都能自由使用的类 Unix 操作系统兼容产品。

# 他有什么应用？

Linux 系统主要被应用于服务器端、嵌入式开发和 PC 桌面 3 大领域，其中服务器端领域是重中之重。全世界多数的公司都在将自己的网站部署在 Linux 系统上。由于其应用的广泛，在作为服务器的领域也使其生态也比 Windows 服务器要丰富得多，也就造成了它的稳定性、实用性、已经工具的丰富程度要比 Windows 服务器要强上很多。当然在桌面领域 Linux 是跟 Windows 的差距还是很大的。

# 他有什么优势？

  • Linux 不仅是免费的，更是开源的，这意味着任何人都可以获得其代码并根据自己的需求进行修改。事实上，这已经孵化出专属 Linux 各发行版的巨大生态系统，我们将在《常见Linux发行版本》一节中展开探讨。
  • Linux 与 Unix 系统兼容，该系统的构建采用了一些与 UNIX 操作系统相同的技术，具备 Unix 几乎所有的优秀特性（系统的稳定性和安全性尤为突出）。
  • 和其他操作系统相比，Linux 更容易安装，且不会收到任何商业化版本的制约。
  • Linux 让开展各种实际有用且具有创造性的事情成为可能。例如，我们可以在一个U盘上装载 Linux 的自生系统引导映像（live boot image），启动一台硬盘已经崩溃的计算机，之后查找并解决该问题。或者，因为 Linux 是一个真正的多用户操作系统，它具有非常好的私密性和稳定性，整个团队可以同时从本地或远程登录进行工作。
  • Linux 提供了复杂的软件包管理系统，可以放心地安装和维护每一个在线资源库中的软件应用。

# Linux 的发行版

Windows 拥有 Windows XP、Windows 7、Windows 8、Windows 10 等众多版本，像 Windows 一样 Linux 也有很多的发行版，只不过其数量远远多于 Windows ，目测有上千个发行版，可参考 Linux 发行时间线 (opens new window)。

# 常见的发行版

目前在国内最常见的发行版本有两个，以 Red Hat 为代表的 Red Hat Linux (收费)、CentOS 这两个发行版；还有一个是以 Debian 为代表的 Debian、Ubuntu 这两个发行版。最开始学习以稳定可靠为基础，推荐使用 CentOS 或者 Ubuntu 进行学习，后续可任意更换喜欢的发行版进行使用。
