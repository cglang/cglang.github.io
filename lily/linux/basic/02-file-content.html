    # 文件和目录管理

    在 Linux 操作系统中，所有的文件和目录都被组织成以一个根节点“/”开始的倒置的树状结构，如下示例。

    /
├── bin
├── boot
├── etc
├── home
│   └── cglang
├── lib
├── srv
├── sys
│   ├── block
│   ├── bus
│   ├── class
│   ├── dev
│   ├── devices
│   ├── firmware
│   ├── fs
│   ├── kernel
│   └── module
├── tmp
├── usr
└── var

    # ls 命令

    我们可使用 ls 命令来查看某目录下的文件以及文件夹

    ls              # 查看当前工作目录
ls /            # 查看根目录
ls -all         # 查看目录下包括隐藏文件和目录
ls -list        # 查看当前目录的的列表形式
ls -l           # ls -list 的其简写
ll              # ls -list -all 的简写

    其他用法可使用 ls --help 查看详细

    # cd 命令

    使用 cd 命令进行工作目录的切换

    cd /            # 切换到 / 目录
cd ~            # 切换到用户(home)目录
cd ..           # 切换到上一层目录树

    同样可使用 cd --help 查看其他用法

    # pwd 命令

    查看当前工作目录

    pwd             # 查看当前工作目录

    提示

    在以下操作中要经常使用 ls 命令查看对文件以及目录的操作，并且所有操作要在用户目录下进行操作。使用 cd ~ 切换到用户目录。

    # mkdir 命令

    mkdir 命令用来创建目录

    mkdir name              # 在当前目录下创建一个名为 name 的目录
mkdir {a，b}             # 创建两个目录a和b
mkdir -p {c/c1，d/d1}    # 创建四个目录c以及c下的c1，目录d以及d下的d1

    创建完可使用 ls 命令进行查看，

    提示

    在 Linux 中文件以 . 开头默认为隐藏文件，必须用 ls -a 才可查看.

    # touch 命令

    touch 用于创建文件

    touch filename          # 创建一个名为filename的文件
touch {f1，f2}           # 一次创建多个文件

    # rm 命令

    删除命令

    rm -d a                 # 删除目录a
rm -rd c                # 递归删除 即删除c目录以及其目录下的所有文件
rm -f filename          # 删除文件filename
rm -r *                 # 删除当前目录下的所有目录及文件

    警告

    千万不要在 root 权限下执行 rm -rf / 命令，这样会删除掉整个系统.

    # cp 命令

    复制命令

    不出意外地话刚才所建的所有文件及目录都在刚才学习rm的时候都给删除了，我们再重新建就好了.

    # 复制执行以下命令
mkdir -p {a/a1，b/b1} && touch {a/a1/{fa1，fa2，fa3} && touch filename}

    cp -f cp -f a/a1/fa1 b/b1/  # copy a/a1/fa1 文件到 b/b1/ 目录下
cp -r a b/                  # copy a 目录及以下所有文件到 b/ 目录下

    # mv 命令

    移动和修改文件名称

    mv a b/b1/                  # 移动 a 目录到 b/b1/ 目录下
mv b/b1/a .                 # 再将 a 目录移动到当前目录下
mv a aa                     # 将目录 a 修改为 aa
mv filename name            # 将文件 filename 修改为 name

    至此几乎关于文件和目录的常用操作就完成了，总就这几个命令

    • ls 查看目录及文件
    • cd 切换工作目录
    • pwd 查看当前工作目录
    • mkdir 创建目录
    • touch 创建文件
    • rm 删除文件及目录
    • cp 复制文件及目录
    • mv 移动文件及目录或者修改文件及目录的名称
