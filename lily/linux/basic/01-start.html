# 开始

# 首先要拥有操作系统

  1. 在自己电脑上装 Linux 和 Windows 双系统
  2. Windows 10 专业版可使用 WSL。
  3. 通过虚拟机安装 Linux 操作系统，可使用 VMware、VirtualBox、Hyper-v等进行安装。
  4. 通过阿里云、腾讯云进行购买虚拟服务器。

如果你觉得自己特别能折腾的话可以选择 1 次一点选择 2，再者就是 3，如果你只想单纯的想要学习 Linux ，想把精力花在开发或者其他上面可以选择购买虚拟服务器。

另外如果允许的话我推荐你以后使用WSL这是一个神器。

# 通过虚拟机进行安装

提示

本教程主要以 Ubuntu 发行版为例。

# 准备

  • Ubuntu 系统镜像 下载地址 (opens new window)
  • VMware 虚拟机软件 下载地址 (opens new window)

# 开始安装虚拟机

安装完成VMware之后打开大致是这样一个界面。

点击创建虚拟机或者菜单-文件-新建虚拟机

选择刚刚下载好的镜像文件

这一步其实随便选择就好

设置虚拟机名称以及位置

这一步按自己实际需要自行修改，默认20G完全足够.

点击下一步之后我们点击自定义硬件我们设置网络适配器为NAT模式，处理器和内存自行设置就好，1核1G也足够使用，Linux 不像 Windows 那般占用很多的资源。

接下来点击关闭 完成即可，然后点击开启虚拟机，进行 Ubuntu 系统的安装配置.
