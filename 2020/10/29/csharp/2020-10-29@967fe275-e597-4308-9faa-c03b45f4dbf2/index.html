C# 属性与字段
2020-10-29

字段

概述

  • 字段是在类或结构中直接声明的任意类型的变量。 字段是其包含类型的成员。
  • 通常情况下，应仅对具有 private 或 protected 可访问性的变量使用字段。

字段的命名规范

推荐字段的命名规范为：下划线(_)开头+camel命名法(首单词字母小写)。

属性

属性是相对于字段来说的，字段仅允许类的内部进行访问，属性则是对外开发，在类的外部也可进行访问。

概述

  • 属性是允许类公开获取或设置值的公共方法。
  • get 属性访问器用于返回属性值，而 set 属性访问器用于分配新值。这些访问器可以具有不同的访问级别。
  • value 关键字用于定义由 set 访问器分配的值。

属性的命名规范

  • 大写字母开头的单词或单词组，即帕斯卡（pascal）命名法
