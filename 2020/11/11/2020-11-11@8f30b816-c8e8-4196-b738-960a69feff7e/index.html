EF Core (一) EF的简单使用

EF Core 简介

EF Core 全称 Entity Framework Core 是轻量化、可扩展、开源和跨平台版的常用 Entity Framework 数据库访问技术。EF Core 可用作对象关系映射(O/RM)。

它不仅仅是一个 O/RM 框架来使用，它的功能要比其他数据库访问框架强大的多，值得去深入学习。
